# 🤖 AI Document Chatbot

An AI-powered chatbot that lets you upload documents (PDF, DOCX, TXT) and interact with them through **summaries** and **question answering**.

---

## ✨ Features
- 📄 **Document Uploads**: Supports PDF, DOCX, and TXT files.  
- 📝 **Summarization**: Instantly generate concise summaries of uploaded documents.  
- ❓ **Question Answering**: Ask questions and get context-aware answers from relevant sections of your documents.  
- 🔀 **LLM Toggle**: Switch between document-based answers and direct responses from the LLM.  
- ⚡ **Efficient Retrieval**: Uses FAISS vector store for fast and accurate semantic search.  
- 🎛️ **Interactive UI**: Built with Gradio for a simple and user-friendly interface.  

---

## 🛠️ Tech Stack
- Python  
- OpenAI API  
- LangChain  
- FAISS Vector Store  
- Gradio UI  
- Hugging Face Spaces  

---

## 🚀 Use Cases
- 📚 **Study Assistant**: Upload lecture notes and get summaries or quiz-style questions.  
- 🔬 **Research Helper**: Quickly extract insights from long papers.  
- 🏢 **Knowledge Base**: Interact with company documents or manuals.
